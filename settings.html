📜 Google Apps Script Code

Copy this script and deploy it as a Web App in your Google Sheet (Extensions ➔ Apps Script). Set access to "Anyone".

Code.gs 
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var timestamp = new Date();
    
    // Open the active spreadsheet
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = ss.getSheetByName("AI Masterclass Registrations");
    
    // Create sheet if it does not exist
    if (!sheet) {
      sheet = ss.insertSheet("AI Masterclass Registrations");
      sheet.appendRow([
        "Timestamp", 
        "Name", 
        "Email", 
        "Mobile", 
        "Profession", 
        "City", 
        "Goal", 
        "Source"
      ]);
      // Format headers bold
      sheet.getRange(1, 1, 1, 8).setFontWeight("bold");
    }
    
    // Append the user details
    sheet.appendRow([
      timestamp,
      data.name,
      data.email,
      data.mobile,
      data.profession,
      data.city || "",
      data.goal || "",
      data.source || "Landing Page"
    ]);
    
    // Send Premium HTML Confirmation Email
    var subject = "🚀 Your Seat Is Confirmed - AI Mastery Kickstart";
    
    var htmlBody = 
      "<div style=\"background-color: #f1f5f9; padding: 40px 20px; font-family: 'Helvetica Neue', Helvetica, Arial, sans-serif;\">" +

          "</div>" +
          // Email Body
          "<div style=\"padding: 40px 30px; color: #334155; line-height: 1.7; font-size: 15px;\">" +
            "<p style=\"margin-top: 0;\">Hi <strong>" + data.name + "</strong>,</p>" +
            "<p>Thank you for registering for Parv Infosoft's <strong style=\"color: #3b82f6;\">FREE 2-Day AI Masterclass</strong>. Your registration has been successfully confirmed!</p>" +

            "<div style=\"background-color: #f8fafc; border: 1px solid #e2e8f0; border-radius: 8px; padding: 20px; margin: 25px 0;\">" +
              "<h3 style=\"margin-top: 0; margin-bottom: 12px; color: #0f172a; font-size: 16px; border-bottom: 1px solid #e2e8f0; padding-bottom: 6px;\">🗓️ Event Logistics</h3>" +

    // Send email using MailApp with htmlBody

  } catch(error) {

  }
}

Implementation Guide:

  1. Open your Google Sheet, click Extensions ➔ Apps Script.
  2. Delete any default code, paste this copied script, and click the Save (Floppy Icon).
  3. Click Deploy ➔ New Deployment (top right).
  4. Click the Gear icon, select "Web App".
  5. Configure:
    • Execute As: Me (your gmail)
    • Who has access: Anyone
  6. Click Deploy, copy the **Web App URL**, and update it in your settings form.
